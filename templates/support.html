Frequently Asked Questions

How do I get started with BugzyAI?

Simply sign up for a free account, follow our quick start guide, and you'll be generating tests in minutes. No credit card required for the free tier.

What types of tests can BugzyAI generate?

BugzyAI can generate UI tests, API tests, accessibility tests, performance tests, and more. Our AI adapts to your specific application and requirements.

Is my data secure with BugzyAI?

Yes, we take security seriously. All data is encrypted in transit and at rest, we're SOC 2 compliant, and we never store your source code or sensitive application data.

Can I integrate BugzyAI with my existing CI/CD pipeline?

Absolutely! We provide integrations for GitHub Actions, GitLab CI, Jenkins, and more. You can also use our REST API for custom integrations.
